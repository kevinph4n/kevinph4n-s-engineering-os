A personal engineering knowledge base built to organize everything related to my learning, projects, research, career, and long-term growth as an engineer.

Unlike traditional project repositories, this repository serves as my **Engineering Operating System** — a place where I continuously collect knowledge, document experiences, and build reusable resources throughout my career.

---

## Why this repository exists?

As I progressed through hackathons, projects, university, and research, I realized that valuable information was becoming scattered across notebooks, bookmarks, cloud drives, and random documents.

This repository solves that problem by becoming a single source of truth for everything related to my engineering journey.

Instead of remembering everything, I build systems that remember for me.
